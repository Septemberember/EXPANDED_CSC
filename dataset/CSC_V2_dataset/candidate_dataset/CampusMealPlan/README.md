# CampusMealPlan

Medium-sized main benchmark candidate for CSC, refined TBFV, and fault
localization experiments.

- Shape: layered meal-plan recommendation rules with one small bounded loop.
- Intended size: enough condition and statement density for meaningful top-k
  localization evaluation.
- Return categories: `-1` invalid, `1` minimal, `2` light, `3` standard, `4`
  premium.
